        In HTML, an attribute provides additional information about an element. Attributes are written within the opening tag and control the behavior or appearance of that element. There are two main types of HTML attributes:




        Types of HTML Attributes
            Global Attributes:

            These can be used on any HTML element.

            Core global attributes include:

            id: A unique identifier for the element.

            class: Defines one or more class names for the element (used for CSS and JavaScript targeting).

            style: Specifies inline CSS styles for the element.

            title: Provides additional information about the element, usually shown as a tooltip when hovering over it.

            Element-Specific Attributes:

            These attributes are specific to certain types of HTML elements and provide additional control or data relevant only to those elements.

            Examples of HTML Attributes
            < a href="https://example.com">
            The href attribute specifies the URL of the page the link goes to.

             < img src="image.jpg" alt="Description">
            src specifies the image path; alt provides alternate text if the image cannot be displayed.

            < form action="/submit" method="post">
            action specifies where the form data is sent; method specifies the HTTP method for submitting data.
